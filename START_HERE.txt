OBFUSCATION TECHNIQUES
----------------------

Available techniques (use with -t flag):

  all         - Apply all safe techniques (recommended)
  sections    - Randomize section names (.text -> .xmkpqr)
  timestamp   - Modify PE timestamp
  dos-stub    - Randomize DOS stub message
  junk        - Add random overlay data
  checksum    - Modify PE checksum
  anti-debug  - Add anti-debug markers
  caves       - Fill code caves with junk

USAGE EXAMPLES
--------------

Basic (all techniques):
  python obfuscator.py program.exe

Custom output file:
  python obfuscator.py program.exe -o protected.exe

Specific techniques:
  python obfuscator.py program.exe -t sections timestamp junk

Light obfuscation (no size increase):
  python obfuscator.py program.exe -t sections timestamp

Heavy obfuscation:
  python obfuscator.py program.exe -t all

MAXIMUM obfuscation (1400%+ size increase!):
  python obfuscator_max.py program.exe

Help:
  python obfuscator.py --help
